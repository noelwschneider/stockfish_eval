# React Template 1.0

This template is meant to provide a blank slate for new node.js projects. It uses the following:

<ul>
    <li>Express</li>
    <li>Postgres</li>
    <li>Axios</li>
    <li>React</li>
    <li>React Router</li>
    <li>Redux</li>
    <li>Material UI</li>
</ul>
